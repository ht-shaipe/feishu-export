## 1. 授权问题

### 1.1 App 权限配置

**症状**: 报错 `code=99992402`（字段验证失败）或 `code=230001`（文档不存在）

**检查步骤**:

1. 登录 [飞书开发者后台](https://open.feishu.cn/app)
2. 选择你的应用
3. 进入「权限管理」
4. 确认以下权限已开通并生效：

| 权限名称 | 状态 |
|----------|------|
| 获取知识库空间列表 | ✅ 已开通 |
| 获取知识库节点 | ✅ 已开通 |
| 获取空间信息 | ✅ 已开通 |
| 获取文档内容 | ✅ 已开通 |
| 导出文件 | ✅ 已开通 |
| 文档内容读取（新版） | ✅ 已开通 |

5. **重要**：权限修改后需要「创建版本」并「申请发布」

### 1.2 Token 过期

**症状**: 报错 `code=99991663` 或 `Token expired`

**解决方案**:

```bash
# 重新登录
feishu-export login

# 或清除旧 Token 后重新授权
feishu-export logout
feishu-export login
```

### 1.3 知识库管理员权限

**症状**: 部分文档报错 `code=131006`（权限不足）

**解决方案**:

1. 打开目标知识空间
2. 点击右上角「···」→「空间设置」
3. 进入「管理员」选项卡
4. 添加应用为管理员

## 2. 导出问题

### 2.1 电子表格导出失败

**症状**: 报错 `error decoding response body` 或 `Network error`

**原因**: 直接下载链接可能已过期或被重定向

**解决方案**: 工具已内置自动重试机制，如果仍然失败：
1. 使用 `--resume` 参数重试
2. 检查网络连接

### 2.2 导出超时

**症状**: 导出任务卡住不动

**原因**: 文档过大或飞书服务器繁忙

**解决方案**:
1. 等待超时自动跳过（默认 2 分钟）
2. 减少并发数：`--concurrency 3`
3. 跳过问题文档继续导出

### 2.3 文档数量不完整

**症状**: 导出的文件数量少于预期

**检查**:
1. 是否所有文档都有权限（权限不足会被跳过）
2. 思维导图和幻灯片暂不支持导出
3. 确认 `--exclude` 参数没有排除不该排除的文档

## 3. 格式问题

### 3.1 格式自动降级说明

| 原始类型 | 请求格式 | 实际降级 | 说明 |
|----------|----------|----------|------|
| sheet (Excel) | docx | xlsx | Excel 无法导出为 Word |
| bitable (多维表格) | docx | xlsx | 多维表格无法导出为 Word |
| 特殊格式 docx | docx | pdf | 内部格式不兼容时降级 |

### 3.2 Markdown 转换质量

转换后的 Markdown 会保留：
- ✅ 标题层级
- ✅ 文本格式（粗体、斜体）
- ✅ 列表结构
- ✅ 表格内容
- ✅ 代码块
- ✅ 图片链接

可能丢失的内容：
- ⚠️ 飞书特有的块引用和 callout
- ⚠️ 高级表格样式（合并单元格可能简化）
- ⚠️ 某些嵌入的小程序/第三方组件

### 3.3 图片处理

- **导出 docx/pdf**: 图片嵌入在文件中
- **导出 md**: 图片转为 Base64 或保持远程链接
- **转换工具**: 图片转为 Base64 嵌入 Markdown

## 4. 性能问题

### 4.1 加快导出速度

```bash
# 方法1: 增加并发数（推荐 5-10）
feishu-export export <space_id> --concurrency 10

# 方法2: 使用断点续导（跳过已下载）
feishu-export export <space_id> --resume

# 方法3: 排除不需要的文件夹
feishu-export export <space_id> --exclude token1,token2
```

### 4.2 速率限制

飞书 API 有以下限制：

| 操作 | 限制 | 工具默认行为 |
|------|------|--------------|
| 创建导出任务 | 100次/分钟 | 并发 5，自动满足 |
| 查询导出状态 | 100次/分钟 | 指数退避重试 |
| 下载文件 | 100次/分钟 | 串行下载 |

### 4.3 大文档导出

对于特别大的文档（>50MB），导出可能需要更长时间：
- 耐心等待（最多 2 分钟）
- 导出任务创建后，文件保留 10 分钟

## 5. 错误码参考

### 飞书 API 错误码

| 错误码 | 含义 | 解决方案 |
|--------|------|----------|
| 0 | 成功 | - |
| 99991663 | Token 无效/过期 | 重新登录 |
| 99991400 | 参数错误 | 检查请求参数 |
| 99992402 | 字段验证失败 | 检查 App 权限 |
| 131006 | 权限不足 | 添加管理员权限 |
| 230001 | 文档不存在 | 文档已删除或移动 |
| 1069918 | 文件扩展名不匹配 | 自动降级处理 |

### 工具内部错误

| 错误信息 | 含义 | 解决方案 |
|----------|------|----------|
| `Network error: connection refused` | 网络连接失败 | 检查网络 |
| `Network error: timeout` | 请求超时 | 稍后重试 |
| `error decoding response body` | 响应解析失败 | 使用 `--resume` 重试 |
| `Export timeout` | 导出任务超时 | 大文档需要等待更久 |

## 6. 日志与调试

### 启用详细日志

```bash
# 使用 RUST_LOG 环境变量
RUST_LOG=debug feishu-export export <space_id>
```

### 检查存储状态

```bash
# 查看配置
feishu-export config show

# 查看 Token 状态（JSON 格式）
cat ~/.config/feishu-export/token.json

# 查看导出进度
ls -la ~/.config/feishu-export/progress/
```

### 清除缓存重新开始

```bash
# 清除导出进度缓存（慎用！会重新下载所有文件）
rm -rf ~/.config/feishu-export/progress/

# 清除所有配置（慎用！需要重新登录）
rm -rf ~/.config/feishu-export/
feishu-export login
```
